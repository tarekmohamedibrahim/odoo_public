Create Simple Excel Reports

This module is an example of using the xlsxwriter features to generate Excel output. In this report I collect the number of days taken by employees for specific period. The report counts only the days within the selected period. For example, if an employee has taken a vacation from 2021-08-30 to 2021-09-02, these are 4 days. In the report if you selected the period from 2021-09-01 to 2021-09-30, then the report will count 2 days only for this employee.
