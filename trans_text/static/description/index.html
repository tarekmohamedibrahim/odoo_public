Simple Transit Text

This module allow adding simple transit text to your form view

After Installing the module, add this tag to your form view:

You would see the moving text like this

Widget parameters are as follows:

Explanation:

field : represents a char field on the main table of the form view

direction : is "left", "right", "up" or "down"

bg_color : is one of the bg-names defined in Odoo

tooltip : is a help text appears when hover over the ribbon

speed : text speed, default value is 10
